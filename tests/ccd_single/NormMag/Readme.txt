Tests in this directory test the normalization feature

Test Criteria
=============

* Input and output data (data.dat|data_format.dat) should be the same
* The normalization factors (norm_factor.dat) should be as expected (given a
  data set and a normalization magnitude)
* Plots should contain the original data

* Check for all possible input formats (rre_rmin, rmag_rpha, cre_rim, cmag_cpha)
* Check for resistivity/conductivity model
