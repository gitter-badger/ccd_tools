dd_single.py¶

dd_single.py is used to fit one or multiple SIP spectra using the Debye (or Cole-Cole) decomposition scheme. No smoothing regularization between the spectra is applied.

Command line options¶

Debye Decomposition
Usage: dd_single.py [options]

Options:
  -h, --help            show this help message and exit
  -f FILE, --frequency_file=FILE
                        Frequency file (default: frequency.dat)
  --ignore=IGNORE_FREQUENCIES
                        Frequency id's to ignore, example:
                        '12,13,14'. Starts with index 0.
  --fmin=DATA_FMIN      Ignore frequencies below this value (default: None)
  --fmax=DATA_FMAX      Ignore frequencies above this value (default: None)
  -d FILE, --data_file=FILE
                        Data file (default: data.dat)
  --data_format=DATA_FORMAT
                        Input data format, possible values are: rmag_rpha,
                        lnrmag_rpha, log10rmag_rpha, rmag_rpha,  rre_rim
                        rre_rmim, cmag_cpha, cre_cim, cre_cmim (default:
                        rmag_rpha). "r" stands for resistance/resistivity, and
                        "c" stands for conductance/conductivity
  -n INT, --nr_terms=INT
                        Number of Debye terms per frequency decade (default:
                        20)
  -o DIR, --output=DIR  Output directory (default: results)
  -p, --plot            Plot final iterations (default: False)
  --plot_reg_strength   Plot regularization strengths of final iterations
                        (default: False)
  -i, --plot_its        Plot spectra of each iteration (default: False)
  --silent              Do not plot any logs to STDOUT (default: False)
  --tmp                 Create the output in a temporary directory and later
                        move it later to its destination (default: False)
  --tausel=STRATEGY     Tau selection strategy: data: Use data frequency
                        limits for tau selection data_ext (default): Extend
                        tau ranges by one frequency decade compared to the
                        'data' strategy. Factors can be set for the low and
                        high frequency by separating with a ',': LEFT,RIGHT,
                        e.g. '10,100'
  --norm=FLOAT          Normalize magnitudes to the linear
                        resistivity/resistance value (default: None)
  --plot_lcurve=ITERATION
                        Plot the l-curve for a selected iteration. WARNING:
                        This only plots the lcurve and does not use it in the
                        inversion process. Use -1 for last iteration.
                        default=None
  --max_it=INT          Maximum nr of iterations (default: 20)
  -v, --version         Print version information
  --output_format=TYPE  Output format (ascii, ascii_audit)
  -c INT, --nr_cores=INT
                        Number of CPU cores to use (default: 1)
  --lambda=FLOAT        Use a fixed lambda (integer), default=None

Note
